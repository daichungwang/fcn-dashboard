模型說明

Baseline：理性市場共識（長期定位）

Pure：模型判斷（策略觀點）

Event：市場情緒（短期偏移）

三層模型定義

Baseline = 理性市場共識（Rational Market Consensus）

Pure = 不含新聞事件的模型判斷

Event = 加入市場事件後的結果

統計解釋

ΔPure = Pure - Baseline

ΔEvent = Event - Pure
