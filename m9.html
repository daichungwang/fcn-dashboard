M9｜資產管理系統

專業資產負債表 / 現金流管理 / 圖表分析 / localStorage 自動保存 / JSON 匯入匯出

本頁定位

  • 把 FCN、股票、存款、負債、生活費、收入放在同一頁看全貌。
  • 左側是輸入區，可新增 / 刪除 / 擴充欄位內容。
  • 右側是 dashboard、圖表與分析建議，會隨數字即時更新。
  • 資料會自動儲存在瀏覽器，重新開啟頁面也會保留。

建議使用流程

  • 先輸入 FCN、股票、存款、負債。
  • 再補上主要日常支出與每月收入。
  • 看淨資產、負債比率、每月淨現金流是否健康。
  • 透過 M2 / M6 持續管理 FCN 與接股後部位。
