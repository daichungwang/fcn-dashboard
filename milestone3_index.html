振宇 FCN 系統｜Milestone 3

Milestone 3 聚焦在 M6 股票庫存與操作策略。
目標是把「接股後的觀察 / 評估 / 策略提醒」獨立成一個可操作頁面，方便你從 FCN 風險管理延伸到持股後續處理。

本次里程碑目標

  • M6 正式納入入口頁面
  • 接股後可快速看到持股總覽
  • 補強技術評估與市場評論
  • 建立技術提醒與 Market Comment 閱讀區塊
  • 讓 M2 → M6 的操作路徑更明確

目前 Milestone 3 主線

  • M2：FCN 持倉健康 / 提早出場 / 到期專區
  • M6：接股後股票追蹤與操作策略
  • M7：股票遴選評定
  • M8：詢價 / 定價模型
  • 逐步形成「選股 → 做單 → 持倉 → 接股後管理」完整流程

  • Worst-of / 健康 / 觀察 / 危險

M6｜股票庫存與操作策略

Milestone 3 重點
接股後的操作面板。把股票庫存、成本、技術位置、策略觀察與市場評論集中在同一頁， 讓你判斷要續抱、觀察、減碼、分批處理，還是等待反彈。
  • 持股總覽：股數、成本、現價、市值、損益與批次資訊
  • 技術評估：結構位置、漲跌表現、波動、趨勢觀察
  • 市場評論：個股與市場背景的綜合說明
  • 技術提醒：過熱 / 反彈 / 壓力 / 風險提示
  • Market Comment：方便快速閱讀的核心評論區
