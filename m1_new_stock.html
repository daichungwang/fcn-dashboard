股票查詢功能

主模式：輸入 1~5 檔股票代碼（每格可輸入完整代碼），系統會先顯示 Basket Summary，再列出個股研究卡。未輸入時，不顯示研究結果。

頁面定位

這一頁不是選股列表，而是研究入口。查 1 檔時看單一公司報告；查 2~5 檔時先看 basket 結構，再看 individual cards。適合 FCN basket 研究與 weak link 判斷。

資料來源

1. data/m1/universe_150.json：股票名單 / category / sector / style
2. data/m1/competitive_cards.json：公司研究主檔
3. data/m1/m1_market_runtime.json：價格 / 報酬 / volume ratio
4. data/m3/m3_output.json：主觀判斷補充
5. data/m7/m7_new_stock_today.json：估值 / 趨勢 / 時機補充
