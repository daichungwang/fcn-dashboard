A正常 EPS 模式

高可信：公司 EPS history + forward EPS / EPS regression 產生 CC；可直接進 M1 排序。

BPartial EPS 模式

中高可信：有部分 EPS / forward EPS，但 analyst_count 或 forward coverage 不完整；可用，但需看 warning。

CRuntime Fundamental Fallback

中可信：EPS 結構資料不足，改用 market_runtime 的 PE / margin / FCF / ROE / debt 等欄位補 CC。

DGlobal / ETF / Proxy

低可信：沒有足夠 EPS，也沒有足夠 runtime fundamentals；或 ETF / global proxy。只能參考，不可視為公司 EPS。
