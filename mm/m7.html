• M7_v2 = 0.55 valuation + 0.30 trend + 0.15 money
• m7_raw_score = 0.30 valuation + 0.25 trend + 0.20 structure + 0.15 timing + 0.10 money
• m7_final_score = clamp(raw + z_adj + h_adj, 0, 10)
• Compare layer 是 statistical adjustment（不是 ranking engine）
• Trend = mean(weekly returns) as long-horizon drift proxy
• Structure = weekly log(price) regression channel health（diagnostic only）
• structure_score = best_structure_r2 × 10（best of linear / quadratic / logarithmic）
• Timing = short swing D1~D5 + 1W（diagnostic only）
• Structure is diagnostic only because valuation already captures much of business/quality/channel expectation.
• Timing is diagnostic only because M7_v2 is intended as a stable stock selection score, not short-term trading score.
• Confidence = coverage + missing fields + warning severity synthesis
