【資料來源】 1. 股票基礎資料：data/pool30.json - symbol / name / sector / subsector / category - 這是股票的基礎分類資料 2. 股票價格 / 報酬：data/stock_runtime.json - price_now - ret_1d / ret_1w / ret_1m / ret_6m / ret_12m 3. 股票評分：js/core/stock_engine.js 4. FCN 評分：本頁內建公式（依今天確認版本） 5. localStorage.runtime_cache_v1 - 若有同股票資料，優先覆蓋最新值 ──────────────────────── 【股票層公式】 Pure Stock Score = Baseline Score + Vol Score 中期波動 = 0.1×|1M| + 0.3×|6M| + 0.6×|12M| 短期波動 / Snapshot Momentum = 0.6×1D + 0.3×1W + 0.1×1M Event Stock Score = Baseline Score + Vol Score + Snapshot Score + (0 × Event Score) = Pure Stock Score + Snapshot Score 註： 目前 M1 的 news / event 尚未正式納入， 所以 Event / News Score 先顯示但不進總分。 ──────────────────────── 【Worst-of / Worst group】 2–3檔 → 最差1檔 4檔 → 最差2檔 5檔 → 最差3檔 ──────────────────────── 【SRI（結構風險）】 SRI = 0.6 × Worst-of penalty + 0.4 × ASSY penalty Category penalty： core → worst 3 / assy 2 defensive → worst 2 / assy 1 growth → worst 2 / assy 0 income → worst 1 / assy 0 speculative→ worst -2 / assy -2 ──────────────────────── 【Rate Score】 <10 → -999 10–12 → -4 12–15 → -2 15–16 → 0 16–18 → 3 18–20 → 5 20–24 → 8 >=24 → 10 【Period Score】 <=3 → 5 <=6 → 2 <=9 → -2 <=12 → -5 >12 → -999 【P-Risk（Gap = Strike - KI）】 gap = 0 → 5 gap < 10 → -7 gap = 10 → 5 gap <= 13 → 4 gap <= 15 → 3 gap <= 18 → 0 gap <= 20 → -4 gap <= 22 → -5 gap < 25 → -8 其餘 → -999 【EKI Bonus】 EKI = 2 非 EKI = 0 ──────────────────────── 【FCN 層公式】 Avg Pure Stock = 所有股票 pure_stock_score 平均 Avg Event Stock = 所有股票 event_stock_score 平均 Pure FCN = 0.4 * Avg Pure Stock + 0.2 * Rate Score + 0.1 * Period Score + 0.1 * P-Risk Score + 0.1 * SRI + EKI Bonus Event FCN = 0.4 * Avg Event Stock + 0.2 * Rate Score + 0.1 * Period Score + 0.1 * P-Risk Score + 0.1 * SRI + EKI Bonus Delta FCN % = ((Event FCN - Pure FCN) / |Pure FCN|) * 100 Delta Label >100 → 非常甜 >50 → 偏甜 >=-20 → 合理 >=-50 → 偏貴 其餘 → 很貴
